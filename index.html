Flappy Capybara

by dowinterfor6

Description

A flappy bird clone with a twist---flappy capybara! With basic game mechanics implemented and two different background music, as well as a looping background, this simple game was built to experiment with applications of Javascript and animation with HTML canvas. Difficulty has been adjusted to attempt to match that of the actual game, though hitbox remains a little bit larger than the shown capybara.
